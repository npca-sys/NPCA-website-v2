サイトリニューアル開始 - 2009/11/27

管理者:Mine

ついに、サイトリニューアルです！といっても今から作るところですが。部内構造改革に伴い、サーバーマシンの役割も重要になってきて、インターネット向けのサービスをこのNPCA-BIG-SERVER(Linux)で、部内向けのサービス(主にActiveDirectory)をこれまでクライアントとして利用してきたマシンで行うことにしました。そして、インターネット向けのサービスには情報管理システムも入るわけです。これは部の情報を一元管理するというもので、それで部の運営情報を部員全員がすぐに把握できるようにするというものです。これを導入するに当たってサーバーマシンをいじらないといけないわけで・・・・。
そして、そのシステムを導入するのにあわせてこれまでHTMLで書いてきたサイトをXHTMLに変更、デザインも一新しようとリニューアルを進めています。といっても、多分四月になって部内の追加購入するものを買ってからになると思いますが・・・・。ついでに、この話って例のごたごたでの改革チーム(顧問、部長、会計など)の主要なメンバーの間ではちらほら出ていたような気も・・・・あんまり覚えていない。最近物覚えが悪いんで・・・・。冗談なしに。
結構今、自分が何をするためにここに来た？とか思うことが多いんですよ、仕事が順調にこなせないほど。さすがに記憶障害だとは思わない(というより思いたくない)が、課題と期日を書いていく手帳(ToDoの紙バージョン？)を購入し逐一記入することで何とか効率も上がりました。ついてに書いてて思ったこと。俺って勉強以外でやたら忙しい気がする。勉強に関してはスケジュール表を作って(といっても毎日のように予定を変更してる気がするが)いるが、それ以外に関しては記憶を頼りにやってきたためやり忘れとかがあって、期日直前や期日後に気がつくとかよくあって・・・・。
で、新しいサイトを書くのはもちろん僕がやるのですが、ここはかっこよくFlashでCGを流してド派手にやるか、XHTMLオンリーでおとなしくこれまでどおりやるか・・・・。前者だと確かにかっこいいかもしれないがFlash入れてない環境だと動かない、動画でやるとアクセス数が多いのでそれのダウンロードで、ネットワークリソースが不足する、ページ開いた途端にド派手なCGはあれかな？ってのもあって迷ってます。かといって後者だと普通すぎて・・・・。う～～ん。何かアドバイスもらえますか？一応、最初にXHTMLで完成させて、後からその辺の派手なコンテンツを足していく方針で行きたいです。最初にXHTMLで作っておけばFlash対応じゃない人はそちらにまわすようにすれば良いですし。もし、これに関して何かアドバイスしてくれる人がいたら連絡からお願いします。

部内のごたごた - 2009/11/22

管理者:Mine

あれから二学期に入って部でごたごたが起きていろいろあったんですね。詳しくは言いませんが。おかげで僕も色々な対処をすることになるわけですよ。部員少ないし。で、11月に入ってやっと落ち着き始めて、気がつくとサーバーメンテナンスを行っていない、まあ、そもそも電源すらまともに投入できて無いわけですよ。一応電源やネットワークの構築もすんで、電源を投入。ちょうど良い機会なのでぐちゃぐちゃになっていたサーバーの中身を肥大化する前に最適化と言うことで再インストール。現在過去の運用形態を維持しながらも管理しやすいようにいろいろやってるところです。今回重点を置いているのはWebServerを中心とした部内情報管理システムの構築。LAMPをフル活用した部員の情報管理やメールによる連絡システムなどの構築を目指しています。といっても余り重要度の高くない内容なのでいつ出来る、いや、そもそもやるかどうかもわからないですけど、実行環境だけは整えています。で、私事でハンドルネームをボンビー(bombee)からまいん(Mine)に変更したため一部ページの内容が変わりました。まあ、たいした違いはないので基本的にスルーの方向性で････。

またまたサーバー停止 - 2009/08/18

管理者:Mine

またまたすいません。夏休みの09/08/05 09:10にサーバーが停止してしまったんですがサーバー管理者である私が部室に行くことができず復旧できていませんでしたが、昨日(2009/08/17)にサーバーの復旧に成功しました。原因は不明ですね････ただ、予想としてはルーターが停止した可能性が高いですね。いずれＰＣルーターにしたいんですが････サーバーを直接WANにつなぐとDHCPのリクエストをプロバイダが受け取ってくれないんです。ルーターと同じ設定なんですけどね････。もしかしてLinuxを排除する気か？それはどうでもいいんだが今後も長期休暇はこうなるかも････僕は名古屋から下宿してるんでね････。いずれ新しい管理者、まあ僕の跡継ぎみたいな後輩が出てくれるとありがたいですね。さすがに高三になってから伝授するのは無理････ついでにいずれ全てのアカウントをこのサーバーで管理する新システムを仮想マシンで実験中でこれに成功したらいやでもサーバー管理者が複数人いないと部が成り立たないですから、何とかしたいですね。
